# 🏠 Home Lab – Security & Infrastructure Projects

This repository contains a collection of hands-on projects built in my home lab environment to strengthen my understanding of security, networking, and system administration.

These projects simulate real-world scenarios where security controls, monitoring, and proper access management are essential.

---

## 🎯 Objectives

- Build practical experience in **security and infrastructure**
- Understand **networking and DNS-level controls**
- Implement **system hardening and service isolation**
- Gain hands-on exposure to **monitoring and logging**
- Apply **least privilege and secure design principles**
